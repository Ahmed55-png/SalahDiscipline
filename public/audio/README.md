# Azan audio files

Place an Azan MP3 here named `azan.mp3` (referenced by the AzanPlayer
component). The file is **not** committed to the repo — add your own.

## Free sources (royalty-free / Creative Commons)

- https://www.islamcan.com/audio/adhan/  (multiple recordings)
- https://archive.org/details/AzanCallToPrayer  (public domain)
- Any famous Qari's Azan recording you have

## File requirements

- Format: MP3 (most browser-compatible)
- Size: keep under ~3 MB so PWA install stays fast
- Length: 2–4 minutes (full azan)

## How to add

1. Download an MP3 you like
2. Rename it to `azan.mp3`
3. Save here as `public/audio/azan.mp3`
4. The app's "Test Azan" button (Profile → 🔊 Azan) will then play it
5. Future: prayer-time auto-play uses the same file
